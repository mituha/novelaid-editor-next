```markdown
## ウィンドウタイトル部分の表示

ウィンドウのタイトル部分には、アプリ名、バージョン、プロジェクト名を表示します。  
また、アプリ名、バージョンについては起動時から表示します。

```
アプリケーション仕様に上記を追加して実装してもらいます。

Electron版ではバージョンを取得するにはルートの`package.json`にもバージョンを追加する必要があります。  

Tauri版では`src-tauri\capabilities\default.json`に権限の追加が必要
```json
  "permissions": [
    ...(略)
    "core:window:allow-set-title"
  ]
```
Tauri版で表示されるバージョンは`src-tauri\tauri.conf.json`の`version`のバージョン。
`tauri.conf.json`の`version`に`../package.json`を指定することで、ルートの`package.json`のバージョンを参照できます。
```
"version": "../package.json"
```

`cargo install cargo-edit` を入れて、`cargo set-version`
