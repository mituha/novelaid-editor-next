# 猫モフ Apps - 小説執筆アプリを創ろう

![](./images/20250622_note_1280_670.png)  

[note](https://note.com/mituha)の方で連載しています。  
ここにおいてあるのは元原稿となります。  
また、未投稿の記事、有料記事部分も含みます。  


猫モフ Apps は、猫をモフモフしながら思いついたアイデアを、バイブコーディングでゆるっと創っていく企画です。  
元々、[Google AI Studio](https://aistudio.google.com/apps)でWEBアプリの雛形を作成、ローカルで[Gemini CLI](https://github.com/google-gemini/gemini-cli)で開発を進めていました。  

そして、[Google Antigravity](https://antigravity.google/)の登場で、更に便利に開発を進められるようになりました。  

![](./images/00/00_novelaid-editor.png)  

そこで作成したのが、自分で使用するための小説執筆アプリ、[novelaid-editor](https://mituha.github.io/novelaid-editor/)です。

以下の記事では、この小説執筆アプリを再実装する形で自分好みのオレオレ小説執筆アプリを創っていきます。
