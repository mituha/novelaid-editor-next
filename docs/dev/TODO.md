## バックエンドのfs用にプロジェクトディレクトリの設定および取得を追加

バックエンドのプラグインのnovelaid-fs用として、以下の２つの機能を追加する。

* プロジェクトディレクトリの設定(novelaid-fs:setProjectDirectory)  
  + プロジェクトディレクトリのパスを引数として、設定する。
* プロジェクトディレクトリの取得(novelaid-fs:getProjectDirectory)
  + 設定されているプロジェクトディレクトリのパスを返す。

現在は内部的に値を保持するのみとします。
ただし、ProjectContextでプロジェクトのディレクトリが設定されるタイミングでバックエンドにも設定するものとします
